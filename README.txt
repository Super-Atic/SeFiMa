SeFiMe
The Simple, intuitive & Secure File Menager

- without MySQL database

Installation:
1 - Download source and unzip on /your/public/root/
2 - Change the secure.php path on index.php
3 - Set user/password at the line in secure.php
4 - Enjoy!
